# PhishGuard

PhishGuard is a Python-based phishing email detection tool. It analyzes suspicious email content and identifies possible phishing attempts using keyword detection, suspicious link detection, scam categorization, risk scoring, and security recommendations.

## Features

- Detects suspicious phishing keywords
- Detects suspicious links
- Identifies tax scams, banking scams, credential theft attempts, and general phishing
- Calculates a risk score
- Shows Low, Medium, or High risk level
- Provides security recommendations
- Saves scan results in a report file

## Technologies Used

- Python
- File Handling
- Rule-Based Detection
- Risk Scoring

## Use Case

This tool can be useful for students, cybersecurity beginners, government offices, financial departments, CA firms, and general users to understand and identify possible phishing emails.

## How to Run

Open the terminal and run:
python3 main.py
